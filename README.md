# bulk-image-resizer
Resizes all the images in a directory to the given width, while maintaining aspect ratio.
## Installation
* Pyinstaller:
`
pyinstaller -w resizer.py
`

## Usage:
* Run in command line or terminal `python3 resizer.py`
